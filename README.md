# Waypoint Employer Portal

Full-stack employer portal focused on the assigned modules:

- Home dashboard
- Employer profile
- Settings
- Notification panel
- Profile editing with persistent backend storage
- Working navigation and logout response
- Job Posting and Event Scheduling placeholders for teammates to integrate

## Run on Windows

1. Install Node.js LTS.
2. Extract the project.
3. Open Command Prompt inside the project folder.
4. Run:

```bash
npm install
npm start
```

5. Open:

```text
http://localhost:3001
```

Data is saved in `data/portal.json` whenever profile or settings are updated.
